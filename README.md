## ✨ Features / 功能特性

- **🎬 Rhythm Analysis / 节奏分析**
  - Average Shot Length (ASL) calculation / 平均镜头时长计算
  - Median Shot Length (MSL) calculation / 中位镜头时长计算
  - Visual rhythm charts / 可视化节奏图表
  - Cutting density analysis / 剪辑密度分析

- **🎨 Visual Analysis / 视觉分析**
  - Frame-by-frame brightness & saturation / 逐帧亮度与饱和度分析
  - Dynamic visual charts / 动态视觉图表
  - Color fingerprint generation / 色彩指纹生成

- **📊 Interactive Charts / 交互式图表**
  - Zoomable timelines / 可缩放时间轴
  - Detailed tooltips / 详细数据提示
  - Dark/Light mode support / 深色/浅色模式支持

- **⚡ High Performance / 高性能**
  - Efficient EDL processing / 高效 EDL 处理
  - Local video analysis / 本地视频分析
  - Privacy-first (all processing happens locally) / 隐私优先（所有处理均在本地进行）

## 🚀 Getting Started / 快速开始

### Online / 在线使用
Visit [https://LuN3cy.github.io/CineViz/](https://LuN3cy.github.io/CineViz/)

### Local Development / 本地开发

1. **Clone the repository / 克隆仓库**
   ```bash
   git clone https://github.com/LuN3cy/CineViz.git
   cd CineViz
   ```

2. **Install dependencies / 安装依赖**
   ```bash
   npm install
   ```

3. **Start development server / 启动开发服务器**
   ```bash
   npm run dev
   ```

4. **Build for production / 构建生产版本**
   ```bash
   npm run build
   ```

## 🛠 Tech Stack / 技术栈

- **Core**: React 19, TypeScript, Vite
- **UI**: Tailwind CSS, Lucide React
- **Visualization**: Recharts, D3.js
- **Desktop**: Tauri (Optional for local exe)
